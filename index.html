About Me

I am a Computer Science student at the University of Zambia with a strong interest in software and web development. My journey started with basic HTML pages where I practiced structuring content using headings, paragraphs, and links. As I progressed, I learned CSS to control layout, spacing, colors, and responsive design so websites work well on both mobile devices and laptops. I currently build web applications using Node.js and Express, focusing on server-side logic, APIs, and dynamic functionality. Alongside web development, I am also learning Android development with Kotlin and Jetpack Compose to build modern mobile applications with clean user interfaces. I enjoy breaking complex problems into smaller parts so I can understand and implement each step clearly. This approach helps me write organized and maintainable code. My goal is to become a well-rounded full-stack developer who builds scalable web systems and useful mobile applications that solve real-world problems.
